Install on your phone

Android (Chrome): Tap the menu ⋮ (top right) → Install app or Add to Home screen. You can also use the Install app button above when Chrome offers it.

Chrome desktop: Use Install app in the header or the install icon in the address bar when the site is installable.

Hourly reminders

Allow notifications so Android can remind you once each hour to log your activity. You must tap the button below—Chrome will ask for permission.

Android: If you blocked alerts before, open Chrome → site info (lock) → Permissions → Notifications for this site, or Settings → Apps → Chrome → Notifications.
